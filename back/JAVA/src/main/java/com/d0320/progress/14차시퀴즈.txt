1)
모든 클래스는 Object를 상속받는다. 
Object형의 객체는 모든 클래스로 다운캐스팅이 가능하다 
그러나 실제 객체를 고려하여 다운캐스팅해야 한다.  ClassCastException  발생 


퀴즈)

Object
   -A (extends Object) 
     놀기()
   -B (extends Object)
     공부하기()  

Object o = new A();
A a =  (A) o;    --->1번
B b  = (B) o;    --->2번

1, 2번 중에서 오류가 발생하는 코드는 ?

답 : 2번 (ClassCastException 발생. 실제 생성된 객체는 A타입이므로 B타입으로 다운캐스팅할 수 없습니다.)
 

상속관계에서 업캐스팅과 다운캐스팅 (형변환)

2)다음에 제공되는 매서드를 호출하시오

class A{
    public static void  매서드A(){   System.out.println("매서드  !!  A")}
}

class B{
   public  void  매서드B(){   System.out.println("매서드 ^^ B")}
}

사용법에 다른 이유를 설명하시오

답 :
A.매서드A(); // static 매서드이므로 클래스명으로 직접 호출 가능 (메모리 static 영역에 로드됨)
B b = new B(); b.매서드B(); // 인스턴스 매서드이므로 객체를 생성(new)한 후 참조변수를 통해 호출해야 함 (Heap 영역에 생성됨)
