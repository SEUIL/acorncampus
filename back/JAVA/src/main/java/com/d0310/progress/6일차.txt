6일차 
 1. new의 의미를 알고 배열을 선언할 수 있다
    - 의미: 'new'는 힙(Heap) 영역에 객체를 생성하기 위해 메모리를 할당하고 그 주소값을 반환하는 연산자입니다.
    - 예시: int[] arr = new int[5]; // 5개의 정수를 담을 공간을 힙에 할당

 2. 기본형변수와 참조형변수를 구분할 수 있다
    - 기본형(Primitive): int, double, boolean 등 실제 데이터 값을 스택(Stack)에 직접 저장합니다.
    - 참조형(Reference): String, 배열, 클래스 등 객체의 메모리 주소(해시코드)를 저장합니다.
    - 예시:
      int age = 20; // 기본형 (값 20 저장)
      String name = new String("Kim"); // 참조형 (주소값 저장)

 3. 1차원배열의 메모리 구조를 알 수 있다
    - 구조: 스택 영역에는 배열을 가리키는 참조 변수가 생성되고, 힙 영역에는 실제 데이터들이 연속된 공간에 할당됩니다.
    - 예시: int[] scores = {10, 20, 30};
      (Stack: scores 변수 -> Heap: [10][20][30] 시작 주소)

 4. 2차원배열의 메모리 구조를 알 수 있다
    - 구조: '배열의 배열' 구조입니다. 행을 관리하는 참조 배열이 먼저 생성되고, 각 요소가 실제 데이터를 가진 하위 배열의 주소를 가리킵니다.
    - 예시: int[][] matrix = new int[2][3];
      (matrix -> [행0주소][행1주소] -> 각각 [0][0][0] 데이터 공간 연결)

 5. 정적메모리와 동적메모리를 구분할 수 있다
    - 정적(Static/Stack): 컴파일 시점에 크기가 결정되며, 메서드 종료 시 자동으로 소멸합니다. (기본형 변수, 참조 변수 등)
    - 동적(Dynamic/Heap): 실행 시점(Runtime)에 'new'를 통해 할당되며, 가비지 컬렉터(GC)가 관리합니다. (배열, 객체 등)

 6.  배열을 사용하여 학생정보를 등록할 수 있다
 7.  배열을 사용하여 학생정보를 변경할 수 있다
 8.  배열을 사용하여 학생정보를 삭제할 수 있다
 9.  배열을 사용하여 학생정보를 조회할 수 있다
 10.  배열을 사용하여 간단한 관리프로그램을 작성하는 경험을 갖는다
